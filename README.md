# Quiz de C
## Sobre o projeto 
Este repositório contém a implementação de um Quiz de C, desenvolvido como projeto final da disciplina APC da UnB

O objetivo principal nesse projeto, foi fazer algo interativo e educativo para os alunos de APC, e também utilizar conceitos de programação, como: estruturas, arrays, strings, funções, laços, manipulação de strings e etc.

## Funcionalidades

  *Iteração de perguntas
  *Coleta de respostas
  *Verificação e pontuação

## Como foi feito
Foi feito com a linguagem C, pelo GDB online
